# Supply Chain Analytics Report & Dashboard

## 📌 Executive Summary
This project delivers a data-driven solution to optimise supply chain performance by identifying procurement bottlenecks and refining demand forecasting models. By combining advanced statistical analyses in Excel with dynamic business intelligence in Power BI, this project translates complex data into prescriptive strategies for safety stock optimisation and continuous process improvement.

## 🛠️ Technical Stack & Methodology
* **Business Intelligence:** Power BI (Data modelling, DAX, R, interactive report design).
* **Advanced Analytics (Excel):** Conducted statistical analyses and testing such as **ANOVA**, **t-Tests**, and **Linear Regression** to isolate variables driving supplier delays.
* **ELT:** Loaded and transformed disparate datasets using Power Query for baseline metrics.

---

## 💡 Key Insights & Prescriptive Recommendations
Based on the analytical report included in this repository, the following strategic actions were recommended:
* **Demand Forecasting:** Utilised linear regression and historical trend analysis to build a robust forecasting model for reducing variance between projected and actual inventory requirements.
* **Bottleneck Mitigation:** Isolated a major 14-day lead time variance within a specific supplier tier using ANOVA testing, recommending a dual-sourcing framework.
* **Procurement & Inventory Optimisation:** Recalculated safety stock levels and reorder points using demand variance analytics to mitigate stockout risks while minimising holding costs across the supply chain.
